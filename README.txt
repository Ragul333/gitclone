This is a clone of github , which will display its users , profiles and its repositories.
1.First a pop-up arises, we should enter a proper username of a github user .
2.It displays repos list in left side of the window,
if we click that repo we will be redirected to another page where we should enter the username again to view the users profile.
3.To find the users click on top left searchbar and press enter key it will redirect to another page from that you can search users by searching names on the search bar.
